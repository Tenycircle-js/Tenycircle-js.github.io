Tenycircle.jsの導入方法

htmlタグで読み込む

ダウンロードして読み込む方法

<script src="path/to/Tenycircle.min.js"></script>

公式cdnを使う方法

<script src="https://tenycircle-js.github.io/cdn/js/Tenycircle.min.js" integrity="sha384-4nX/dN0vDO/6v6NOwb986md5tUkB8okB0rmswUaXNk+t8dylckglmhzMx/QDN1n1" crossorigin="anonymous"></script>

ちょっと整形

<script
    src="https://tenycircle-js.github.io/cdn/js/Tenycircle.min.js"
    integrity="sha384-4nX/dN0vDO/6v6NOwb986md5tUkB8okB0rmswUaXNk+t8dylckglmhzMx/QDN1n1"
    crossorigin="anonymous"
></script>

import

Tenycircle.mjsのTenycircle.mjsのcdnを読み込む

全てをインポート

import * as $ from 'https://tenycircle-js.github.io/cdn/mjs/Tenycircle.mjs'

限定的にインポートする

import { funcname } from 'https://tenycircle-js.github.io/cdn/mjs/Tenycircle.mjs'

※funcnameはインポートしたい関数の名前を入れてください
