all-in-packとは?

all-in-packとは、Tenycircle.jsと拡張機能を含んだパックであり、
一括で制御できます。

使用方法

githubからファイルをダウンロードし、
<script src="pass/to/Tenycircle.all.min.js"></script>
または、このページからcdnを取得する方法もあります
<script src="https://Tenycirclejs.github.io/cdn/all/TenyCircle.all.min.js"></script>

初期化方法

初期化しないとなにもできませんので $.init関数を使います
例えば 全て(拡張機能込)を使うなら

  $.init(function(){
  $.init().useall()
}) のように書きます

また、Tenycircle.jsだけ使う時は、

  $.init(function(){
  $.init().usemain()
}) のように書きます

拡張機能も使う時は

  $.init(function(){
  $.init().usemain()
  $.init().useex(拡張機能名(instanceやDOMなど))
}) のように書きます
